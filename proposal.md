# Project Proposal

## 1. Project Title

**LMS-Aware AI Copilot for Personalized Learning**

## 2. Project Objectives

**Main Objectives:** Build an AI-powered assistant that personalizes the user's study process by integrating with LMS (e-Class) data.

- **Objective 1 (Front End):** Implement a seamless Chrome Extension Sidebar that overlays on e-Class pages without disrupting the user's view.
- **Objective 2 (Back End):** Build an integrated system for LMS data scraping (Session-based) and a centralized database for course materials and personal logs.
- **Objective 3 (Back End2):** Develop a Multi-Document RAG engine to deliver hyper-personalized feedback and Q&A based on course materials (PDFs, PPTs) and lecture logs retrieved from the LMS.
