Ingest Pipeline — Cookie → SQLite

JSESSIONID 하나로 세션을 확립하고 KJKEY 단위로 과목 컨텍스트를 설정.
강의자료는 다운로드 → 텍스트 추출 → 페이지 단위 청크 분할 → Doc_Chunk 저장.

Chat / RAG Flow — FTS5 Keyword → Claude

질문에서 키워드를 추출해 SQLite FTS5로 Doc_Chunk를 검색.
chunk_id · material_id · page_ref로 출처를 추적하고 Chat_Log의 sources 필드에 JSON으로 저장.
피드백(👍/👎)은 Chat_Log.feedback_score에 즉시 반영.

DB Chat_Session user_id · course_id · created_at DB Chat_Log role · content · sources · feedback_score sources [{chunk_id, material_id, page_ref}] FTS5 Doc_Chunk MATCH keywords ORDER BY rank LIMIT k
