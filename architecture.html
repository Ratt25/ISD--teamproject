End-to-End System

Chrome Extension이 쿠키를 포착해 FastAPI로 전달하면 Scraper가 e-Class에서 강의자료·활동을 수집합니다.
수집된 자료는 청크로 분할되어 SQLite(FTS5)에 저장되고, 질문 시 키워드 매칭으로 컨텍스트를 조합해 Claude가 답변합니다.

Ingest Pipeline — Cookie → SQLite

JSESSIONID 하나로 세션을 확립하고, KJKEY 단위로 과목 컨텍스트를 설정한 뒤
강의자료를 다운로드·청크 분할해 Doc_Chunk에 저장합니다. MD5 체크섬으로 중복을 방지합니다.

DB User lms_id · enc_session_cookie · last_sync_at DB Course lms_url_id(=KJKEY) · course_code · title DB Material title · file_type · file_path · checksum DB Doc_Chunk content · page_ref · chunk_index DB Learning_Activity title · status · due_date

Chat / RAG Flow — Keyword FTS → Claude

질문에서 키워드를 추출해 SQLite FTS5로 관련 Doc_Chunk를 검색합니다.
chunk_id · material_id · page_ref로 출처를 추적하고, 답변과 함께 Chat_Log에 JSON으로 저장합니다.

DB Chat_Session user_id · course_id · created_at DB Chat_Log role · content · sources · feedback_score sources JSON [{chunk_id, material_id, page_ref}] FTS Doc_Chunk MATCH keywords ORDER BY rank LIMIT k
